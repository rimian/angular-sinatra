# Angular Sinatra

A Sinatra application running AngularJS and RSpec integration specs

This is a demo for integration testing AngularJS applications.

# Requirements

A basic understanding of Ruby, AngularJS and RSpec.

## Developing

```
bundle install
npm install
```

## Testing

```
rspec
```

## Running Locally

```
rackup -p8000 config.ru
```
